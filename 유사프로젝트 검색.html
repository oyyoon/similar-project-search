📖 사용설명서 — 키 발급 · 사용법

이 도구는? 공공데이터(data.go.kr)로 유사 공사를 검색합니다. 상단 탭으로 🏢 건축물대장(실제 착공~사용승인)과 🏗️ 리모델링 계약정보(나라장터 계약상)를 오갑니다.

🔑 서비스 키가 필요합니다. 키는 파일이 아니라 이 브라우저에 저장되어 처음 한 번만 넣으면 됩니다. 얻는 방법 두 가지:

[방법 A · 권장] 본인 키 직접 발급 (무료)

  1. data.go.kr 가입·로그인
  2. API 「활용신청」(자동승인): 🏢 건축물대장정보 서비스 · 🏗️ 나라장터 계약정보서비스 (두 탭 모두 쓰려면 각각 신청)
  3. 마이페이지 → 개발계정에서 일반 인증키 확인
  4. 위 서비스 키 칸에 붙여넣기 (자동 저장)

Decoding 키? 인증키가 영문·숫자만(예: b372f7…546e9)이면 그대로 복사. %2B·%2F 같은 기호가 있으면 「Decoding」 값 사용.

[방법 B] 제공자에게 받은 키 사용 — 받은 키 문자열을 위 서비스 키 칸에 붙여넣기. ⚠ 키는 비밀번호와 같고, 하루 한도(10,000회)를 함께 나눠 씁니다.

🔎 사용법 — 키 입력 → 조건 입력 → 「검색」 → 표 확인 → 「Excel 다운로드」. 계약정보 탭은 공사분류 체크로 부대공종(전기·소방 등) 제외, 전 차수 펼쳐보기 가능.

❓ 안 될 때 — Unauthorized/거절: 키 또는 활용신청 문제(승인·Decoding 확인). 계약정보 느림: 날짜를 1~2개월씩 좁혀서(받은 기간은 캐시). 인터넷 필요·크롬/엣지 권장·두 파일 같은 폴더 유지.

CORS 오류가 뜨면?
  1. 공공데이터포털 API는 브라우저 직접 호출 시 CORS가 막힐 수 있습니다.
  2. 해결: 크롬을 --disable-web-security --user-data-dir=C:\tmp\chrome 옵션으로 실행하거나, 간단한 로컬 프록시(예: cors-anywhere)를 거치세요.
  3. 또는 아래 「프록시 URL」에 자체 프록시 주소를 넣으면 그 경로로 호출합니다.
