# 유사프로젝트 검색

건축물대장 정보 / 나라장터 리모델링 계약정보를 조회하는 단일 페이지 웹도구입니다.

- **웹사이트 진입점:** `index.html` (자체 포함형 — 두 도구를 탭으로 통합)
- **API 키:** 코드에 포함되지 않습니다. 사용자가 브라우저에 입력하면 localStorage에 저장됩니다.
  공공데이터포털(data.go.kr)에서 발급한 키를 사용하세요.

## 업데이트 방법
`index.html`을 수정한 뒤 아래를 실행하면 웹사이트가 같이 갱신됩니다.

```
git add -A
git commit -m "update"
git push
```

GitHub Pages가 push 후 약 1분 내 자동 재배포합니다.

## 백업/원본
- `launcher-old.html` : 예전 런처(외부 파일 로드형). 현재 미사용.
- `유사프로젝트 검색.html`, `유사프로젝트 검색(리모델링-계약정보).html` : 통합 전 개별 도구 원본.
- `유사프로젝트 검색 (통합본).html` : 통합본 원본 백업(현재 `index.html`과 동일 내용).
